Mathematical Methodology

Each sportsbook posts a line on a player prop. After removing the vig via proportional normalization, this implies a probability pi. We model each book's line as:

p_i = p* + ε_i
where E[ε_i] = 0, Var(ε_i) = σ_i², Cov(ε_i, ε_j) = 0

We want a linear unbiased estimator p̂ = Σ w_i · p_i that minimizes variance, subject to Σ w_i = 1. Setting up the Lagrangian and solving gives the optimal weights:

w_i = σ_i⁻² / Σ_j σ_j⁻²

Achieved minimum variance: Var(p̂) = 1 / Σ_j σ_j⁻²

This is the Gauss-Markov theorem: the precision-weighted estimator is the Best Linear Unbiased Estimator (BLUE). Each book's σ_i² is estimated empirically as the mean squared error of its opening line against Pinnacle's closing line, which the literature establishes as the industry benchmark for true probability (Kaunitz, Zheng & Kreiner, 2017).

An edge is flagged when the best available line implies a probability that diverges from p̂ by more than both a minimum threshold (3%) and the 95% confidence interval half-width — ensuring the signal clears the noise floor of the estimation model.

Limitation: the model assumes uncorrelated book errors. In practice, books correlate because they all observe Pinnacle. Relaxing this requires modeling the correlation structure explicitly — flagged as future work.
